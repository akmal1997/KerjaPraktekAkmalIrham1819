Bagaimana cara memakai aplikasi ini?

Drone Monitoring Apps

  • Pertama, buka http://localhost:5000 atau http://127.0.0.1:5000 pada browser. Lalu klik tombol MENU di bawah

  • Kemudian pada "MENU" klik "Mulai Drone Monitoring" untuk memulai drone tracking.

  • Kemudian muncul halaman koneksi. Masukkan jenis koneksi, IP address, port pada drone (untuk menyambungkan ke Dronekit-SITL simulator, masukkan jenis koneksi TCP, ip address 127.0.0.1, port 5760. Kemudian Masukkan garis lintang dan garis bujur home location dronenya.

  • Tunggu beberapa saat, lalu akan muncul halaman yang memberitahukan bahwa koneksi sukses. Lalu klik "Kembali ke menu utama"

  • Kemudian muncul halaman utama yang menampilkan peta, navbar, dan data telemetri drone.

  • Pada Navbar, terdapat menu:

    • Halaman Utama, untuk kembali ke halaman utama.

    • Cara Pakai, untuk melihat cara pakai aplikasi ini.

    • Monitoring, untuk melihat posisi dan data telemetri drone.

    • Add New Waypoint, untuk menambahkan waypoint baru.

    • Takeoff, untuk menerbangkan drone.

    • Go To, untuk menjalankan drone melalui waypoint yang telah di-set.

    • Landing, untuk mendaratkan drone.

    • Clear Waypoints, untuk menghapus waypoint. Tetapi untuk melakukan hal itu harus mendaratkan drone terlebih dahulu.

    • Disconnect, untuk memutuskan sambungan dengan drone.

  • Untuk menambahkan waypoint, klik tombol Add New Waypoint pada navbar, lalu muncul tampilan seperti di atas. Isi latitude, longi tude, ketinggian, dan kecepatan yang diinginkan. Jika menginginkan kecepatan default, maka masukkan -1 pada kecepatan.

  • Jika sudah menambahkan waypoint, maka daftar waypoint akan tampil di bawah dan pada peta di halaman monitoring. Kemudian jika kita telah selesai menambahkan waypoint, klik Go To untuk memulai perjalanan, atau Main Menu untuk kembali ke halaman monitoring.

  • Untuk take off, klik Takeoff pada navbar. Kemudian muncul halaman seperti di atas. Masukkan altitude yang diinginkan lalu klik Takeoff!

  • Setelah takeoff berhasil, muncul halaman seperti di atas, kemudian klik Kembali ke menu utama.

  • Untuk memulai perjalanan, klik Go To. Pastikan sudah men-set waypoint dan men-takeoff kan drone terlebih dahulu!

    Error yang akan terjadi:

    • Jika belum men-set waypoint, maka akan dialihkan ke halaman waypoint untuk men-set waypoint terlebih dahulu.
    • Jika belum men-takeoffkan drone, maka akan muncul pesan error seperti di atas. Klik Takeoff untuk menerbangkan drone, atau klik Kembali ke menu utama untuk kembali ke halaman monitoring.

    Jika berhasil, maka akan muncul tampilan seperti ini di halaman monitoring.

  • Untuk mendaratkan pesawat, klik Landing. Setelah itu akan dialihkan ke halaman monitoring.
  • Untuk menghapus waypoint, klik Clear Waypoints. Pastikan pesawat telah didaratkan sebelum menghapus waypoint.

    Error yang akan terjadi:

    • Jika belum mendaratkan pesawat, maka muncul pesan error seperti di atas. Klik Daratkan Pesawat Ini untuk mendaratkan drone, atau klik Kembali ke menu utama untuk kembali ke halaman monitoring.
  • Untuk mendisconnect dengan drone, klik Disconnect. Kemudian, akan dialihkan ke halaman utama dan semua waypoint akan dihapus.
